# library_management
## Library management system
* Books(add books, delete books, query books, modify books)
* Roles(readers, Admin)
### As for readers:
  * query books (demostrate all books, query books by books'name)
  * borrow books
### As for Admin
  * query books(demostrate all books, query books by books'name)
  * add books
  * delete books
  * modify books
